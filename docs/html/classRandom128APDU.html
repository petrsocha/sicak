Detailed Description

CPA Measurement scenario plugin: sends N times a command APDU: CLA=0x80, INS=0x60, P1=P2=0x00, Lc=0x10, Le=0x10 with 16 bytes of random data and receives 16 bytes of ciphertexts back. Makes use of multiple captures per oscilloscope run (e.g. Picoscope's rapid block mode).
